Detailed Description

a mathematical expression.

This data structure represents a mathematical expression that might contain variables. variables are represented with a single alphabet letter.

Expression grammar

  • atom = number | identifier, [ "(", [ expression ], { ",", expression }, ")" ] | "(" expression ")"
  • primary = atom, [ "^", factor ]
  • factor = "-" factor | primary
  • term = factor, { ("*" | "/"), factor }
  • expression = term, { ("+" | "-"), term }
